# Content Inventory Rationale

The system uses a story-first model instead of media-first to avoid uncontrolled growth and duplication.

Newspaper layout requires hierarchical control:
- primary stories anchor attention
- secondary support narrative depth
- briefs provide breadth

Separation of concerns:
- D1 = brain (stories, relationships)
- B2 = storage (media)

This ensures searchability, integrity, and scalability.
