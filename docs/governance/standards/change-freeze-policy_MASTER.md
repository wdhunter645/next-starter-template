# Change Freeze Policy (MASTER)

## Purpose
Prevent instability during critical periods.

## Freeze Triggers
- Active outage
- Major deployment window
- High-risk release phase

## Freeze Rules
- No new features
- Stability fixes only
- Rollbacks allowed
