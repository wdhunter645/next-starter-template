# How to Manage Content Inventory

## Add a New Story
1. Define tag
2. Check for existing canonical record
3. Insert story record
4. Link approved media by id

## Add Media
1. Validate relevance
2. Upload to B2
3. Create media record
4. Link to story

## Handle Submissions
1. Insert into submission queue
2. Run automated checks
3. Manual review
4. Approve or reject
