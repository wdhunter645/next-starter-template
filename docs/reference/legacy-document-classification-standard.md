# Legacy Document Classification Standard

## Classification States

### Canonical

Current authoritative source.

### Transitional

Partially active document pending migration or replacement.

### Historical

Retained for project history and context only.

### Deprecated

No longer valid for implementation or governance decisions.

## Rules

Historical and deprecated documents must not silently override canonical authority.

Agents must prefer canonical and transitional authority before using historical planning material.
