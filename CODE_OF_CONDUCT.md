# Code of Conduct

## Our Commitment

The Lou Gehrig Fan Club is committed to providing a respectful,
inclusive, and harassment-free environment for all participants.

## Expected Behavior

- Be respectful and constructive.
- Assume positive intent.
- Focus on improving the project.

## Unacceptable Behavior

- Harassment, hate speech, or discrimination.
- Personal attacks.
- Disruptive or abusive conduct.

## Enforcement

Project maintainers may remove, edit, or reject contributions
that violate this Code of Conduct.
